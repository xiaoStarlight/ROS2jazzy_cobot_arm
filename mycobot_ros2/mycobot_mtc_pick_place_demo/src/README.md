mtc_node.cpp:

该文件实现了主要的moveit任务构造器节点，用于拾放任务，它设置规划场景，创建moveit任务构造器任务，包含多个阶段，例如移动到拾取、抓取、提升、放置，负责处理任务的执行过程。协调机械臂的运动来完成拾取和放置的操作

cluster_extraction.cpp：

该cpp文件里包含了用区域生长算法从点云中提取聚类的函数，它把可能属于同一物体的点分组，帮我们区分场景中不同的物体，提取的聚类可单独处理，用于物体识别或操作任务。

get_planning_scene_client.cpp：

该文件实现了一个测试客户端，用于getplanningscene服务，负责请求当前规划场景，包含环境中物体信息。

get_planning_scene_server.cpp：

该文件实现了gitplanningscene服务的服务器，同时处理点云数据和RGB红绿蓝图像数据，为moveit规划场景生成碰撞物体，这些碰撞物体所代表的就是机器人环境中的各种障碍物和物体，便于实现精确的运动规划和物体操作

normals_curvature_and_rsd_estimation.cpp：

法线、曲率和RSD估计函数，包含了一些用于估算法向量、曲率值和每个点中点云中的半径基表面描述符值的函数，曲率值和基于半径的表面描述符值，在学术文献中常简称为RSD值，这些特征帮助理解场景中不同表面的形状和朝向，估计出的特征可用于物体识别、分割和抓取规划

object_segmentation.cpp：

该文件实现了对输入的3D点云进行对象分割的功能，包含了用于拟合几何基元的函数，比如把圆柱体和盒子转换成点云数据，这些数据之后会被用来识别和表示场景中的对象

plane_segmentation.cpp：

该文件从给定的点云中，把支撑平面和物体分割出来，能识别出物体放置的那个表面，就像一张桌子一样
